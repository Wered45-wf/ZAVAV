# Аренда яхт
Веб-сайт для аренды яхт с системой бронирования, созданный на PHP с использованием Bootstrap

## Осовыне возможности
- Регистрация и авторизация
- Каталог с яхтами
- Система бронирования яхт
- Личный кабинет

## Технологии
- **Frontend**: Bootstrap
- **Backend**: PHP
- **База данных**: MySQL
